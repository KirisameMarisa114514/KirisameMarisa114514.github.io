Vim快捷键

Created2025-11-13|Updated2026-07-20
|Word Count:4k|Reading Time:14mins|Post Views:

Vim 常用快捷键与命令手册

**使用说明：**以下快捷键默认在 Vim 的普通模式下执行。
Ctrl + x 表示按住 Ctrl，再按下 x。

1. 模式切换

1.1 进入不同模式

模式 快捷键 说明
普通模式 Esc 返回普通模式，用于移动、复制、删除和执行命令
插入模式 i 在光标前开始输入
插入模式 a 在光标后开始输入
插入模式 o 在当前行下方新建一行并输入
插入模式 O 在当前行上方新建一行并输入
可视模式 v 按字符选择文本
可视行模式 V 按整行选择文本
可视块模式 Ctrl + v 按矩形区域选择文本
命令行模式 : 输入保存、退出、替换和设置命令
搜索模式 / 从当前位置向下搜索
反向搜索模式 ? 从当前位置向上搜索

1.2 退出当前模式

当前模式 快捷键 结果
插入模式 Esc 返回普通模式
可视模式 Esc 取消选择并返回普通模式
命令行模式 Esc 取消当前命令
搜索模式 Esc 取消当前搜索输入
任意编辑状态 Ctrl + [ 效果通常与 Esc 相同

2. 光标移动

2.1 基础移动

功能 快捷键 说明
向左移动 h 向左移动一个字符
向下移动 j 向下移动一行
向上移动 k 向上移动一行
向右移动 l 向右移动一个字符
移动到下一个单词 w 跳到下一个单词开头
移动到上一个单词 b 跳到上一个单词开头
移动到单词结尾 e 跳到当前或下一个单词末尾

2.2 行与文件移动

功能 快捷键 说明
移动到行首 0 移动到当前行第一个字符
移动到首个非空字符 ^ 跳过行首空格和缩进
移动到行尾 $ 移动到当前行末尾
移动到文件开头 gg 跳到文件第一行
移动到文件结尾 G 跳到文件最后一行
跳转到指定行 nG 例如 20G 跳到第 20 行
跳转到指定行 :n 例如 :20 跳到第 20 行
跳转到匹配括号 % 在成对括号之间跳转

2.3 页面滚动

功能 快捷键 说明
向上滚动半页 Ctrl + u 向文件上方滚动半页
向下滚动半页 Ctrl + d 向文件下方滚动半页
向上滚动一页 Ctrl + b 向文件上方滚动一整页
向下滚动一页 Ctrl + f 向文件下方滚动一整页
光标移到屏幕顶部 H 跳到当前窗口顶部
光标移到屏幕中间 M 跳到当前窗口中间
光标移到屏幕底部 L 跳到当前窗口底部
当前行置于屏幕顶部 zt 滚动窗口，使当前行位于顶部
当前行置于屏幕中间 zz 滚动窗口，使当前行居中
当前行置于屏幕底部 zb 滚动窗口，使当前行位于底部

3. 文本插入与修改

3.1 插入与替换

功能 快捷键 说明
在光标前插入 i 进入插入模式
在光标后插入 a 从光标后开始输入
在行首插入 I 在当前行首个非空字符前输入
在行尾插入 A 在当前行末尾输入
下方新建一行 o 在当前行下方新建并编辑
上方新建一行 O 在当前行上方新建并编辑
替换单个字符 r 替换光标下的一个字符
连续替换字符 R 进入替换模式
修改当前单词 cw 删除到单词结尾并进入插入模式
修改当前行 cc 清空当前行并进入插入模式
修改到行尾 C 删除光标到行尾并进入插入模式

3.2 删除操作

功能 快捷键 说明
删除当前字符 x 删除光标下的字符
删除前一个字符 X 删除光标前的字符
删除当前行 dd 删除整行
删除多行 ndd 例如 5dd 删除 5 行
删除到单词开头 dw 删除光标到下一个单词开头
删除到单词结尾 de 删除光标到当前单词结尾
删除到行尾 D 或 d$ 删除光标到当前行末尾
删除到行首 d0 删除光标到当前行开头
删除到文件开头 dgg 删除光标到文件开头
删除到文件结尾 dG 删除光标到文件结尾

3.3 复制与粘贴

功能 快捷键 说明
复制当前行 yy 复制整行
复制多行 nyy 例如 5yy 复制 5 行
复制单词 yw 复制到下一个单词开头
复制到行尾 y$ 复制光标到当前行末尾
复制到行首 y0 复制光标到当前行开头
复制到文件开头 ygg 复制光标到文件开头
复制到文件结尾 yG 复制光标到文件结尾
在光标后粘贴 p 在光标或当前行后粘贴
在光标前粘贴 P 在光标或当前行前粘贴

3.4 撤销与重复

功能 快捷键 说明
撤销操作 u 撤销最近一次修改
重做操作 Ctrl + r 恢复被撤销的修改
重复上次修改 . 再次执行最近一次修改
合并下一行 J 将下一行连接到当前行
重复命令 n命令 例如 3dd、5yy、10j

4. 搜索与替换

4.1 搜索文本

功能 命令 说明
向下搜索 /文本 输入后按 Enter 开始搜索
向上搜索 ?文本 从当前位置向上搜索
下一个匹配项 n 按原搜索方向查找下一个
上一个匹配项 N 按相反方向查找
搜索当前单词 * 向下搜索光标所在单词
反向搜索当前单词 # 向上搜索光标所在单词
关闭搜索高亮 :nohlsearch 临时清除搜索结果高亮

n 是“继续沿当前搜索方向”，N 是“反向查找”，并不是固定的下一个或上一个。

4.2 替换文本

功能 命令 说明
替换当前行首个匹配 :s/old/new/ 只替换当前行第一个匹配
替换当前行全部匹配 :s/old/new/g 替换当前行所有匹配
替换全文全部匹配 :%s/old/new/g 替换整个文件
全文替换并确认 :%s/old/new/gc 每次替换前进行确认
替换指定行范围 :5,20s/old/new/g 替换第 5～20 行
忽略大小写替换 :%s/old/new/gi 匹配时忽略大小写
替换可视区域 :'<,'>s/old/new/g 替换当前选中的文本

确认替换时常用选项：

按键 说明
y 替换当前匹配
n 跳过当前匹配
a 替换后续全部匹配
q 退出替换
l 替换当前匹配后退出

5. 文件操作

功能 命令 说明
打开文件 :e filename 打开指定文件
重新加载文件 :e! 放弃未保存修改并重新加载
保存文件 :w 保存当前文件
另存为 :saveas filename 将当前内容保存为新文件
保存并退出 :wq 保存后退出
保存并退出 :x 有修改时保存，然后退出
快速保存退出 ZZ 等价于有修改时保存并退出
退出 Vim :q 文件无未保存修改时退出
强制退出 :q! 放弃修改并退出
快速强制退出 ZQ 放弃修改并退出
保存所有文件 :wa 保存所有已打开文件
退出所有窗口 :qa 退出全部窗口
强制退出所有窗口 :qa! 放弃所有修改并退出
保存并退出所有窗口 :wqa 保存全部文件后退出

6. 缓冲区与分屏

6.1 缓冲区操作

功能 命令 说明
查看缓冲区列表 :ls 或 :buffers 显示所有已打开文件
下一个缓冲区 :bnext 或 :bn 切换到下一个文件
上一个缓冲区 :bprevious 或 :bp 切换到上一个文件
切换到指定缓冲区 :buffer n 切换到编号为 n 的缓冲区
删除当前缓冲区 :bdelete 或 :bd 关闭当前缓冲区

6.2 分屏操作

功能 命令 说明
水平分屏 :sp filename 上下排列窗口
垂直分屏 :vsp filename 左右排列窗口
切换到下一个窗口 Ctrl + w，再按 w 在窗口之间循环切换
切换到左侧窗口 Ctrl + w，再按 h 移动到左侧窗口
切换到下方窗口 Ctrl + w，再按 j 移动到下方窗口
切换到上方窗口 Ctrl + w，再按 k 移动到上方窗口
切换到右侧窗口 Ctrl + w，再按 l 移动到右侧窗口
关闭当前窗口 Ctrl + w，再按 c 关闭当前分屏
仅保留当前窗口 Ctrl + w，再按 o 关闭其他分屏
窗口等宽等高 Ctrl + w，再按 = 平均分配窗口大小
增加窗口高度 Ctrl + w，再按 + 增加当前窗口高度
减少窗口高度 Ctrl + w，再按 - 减少当前窗口高度
增加窗口宽度 Ctrl + w，再按 > 增加当前窗口宽度
减少窗口宽度 Ctrl + w，再按 < 减少当前窗口宽度

7. 显示与编辑设置

7.1 界面显示

功能 命令 说明
显示绝对行号 :set number 在左侧显示行号
隐藏绝对行号 :set nonumber 关闭行号显示
显示相对行号 :set relativenumber 显示相对光标的行号
隐藏相对行号 :set norelativenumber 关闭相对行号
高亮当前行 :set cursorline 突出显示光标所在行
取消当前行高亮 :set nocursorline 关闭当前行高亮
显示不可见字符 :set list 显示制表符、行尾等字符
隐藏不可见字符 :set nolist 关闭不可见字符显示
高亮匹配括号 :set showmatch 短暂高亮对应括号
开启搜索高亮 :set hlsearch 高亮全部搜索结果
关闭搜索高亮 :set nohlsearch 禁用搜索结果高亮
开启语法高亮 :syntax enable 启用语法高亮
关闭语法高亮 :syntax off 禁用语法高亮
显示状态位置 :set ruler 显示光标行列位置

7.2 搜索设置

功能 命令 说明
搜索忽略大小写 :set ignorecase 搜索时不区分大小写
智能大小写 :set smartcase 包含大写字母时自动区分大小写
输入时实时搜索 :set incsearch 输入搜索词时即时跳转
高亮搜索结果 :set hlsearch 高亮全部匹配文本

7.3 缩进设置

功能 命令 说明
Tab 转为空格 :set expandtab 插入空格而不是真实 Tab
Tab 保留为制表符 :set noexpandtab 插入真实 Tab
设置 Tab 宽度 :set tabstop=4 Tab 显示为 4 个空格宽度
设置自动缩进宽度 :set shiftwidth=4 缩进操作使用 4 个空格
设置软 Tab 宽度 :set softtabstop=4 编辑时按 4 个空格处理
开启自动缩进 :set autoindent 新行继承上一行缩进
开启智能缩进 :set smartindent 根据代码结构自动缩进
启用文件类型缩进 :filetype plugin indent on 根据文件类型加载缩进规则

indentline 不是 Vim 内置选项，通常需要安装缩进线插件后才能使用。

8. 代码缩进

功能 快捷键 说明
当前行向右缩进 >> 增加一级缩进
当前行向左缩进 << 减少一级缩进
多行向右缩进 n>> 例如 5>> 缩进 5 行
多行向左缩进 n<< 例如 5<< 减少 5 行缩进
可视区域向右缩进 > 在可视模式下使用
可视区域向左缩进 < 在可视模式下使用
自动格式化当前行 == 根据缩进规则格式化
自动格式化全文 gg=G 格式化整个文件
自动格式化选中区域 = 在可视模式下使用

9. 代码折叠

功能 快捷键或命令 说明
启用折叠 :set foldenable 开启代码折叠功能
禁用折叠 :set nofoldenable 暂时关闭折叠显示
手动创建折叠 zf + 移动命令 例如 zf5j 折叠下面 5 行
打开当前折叠 zo 展开光标所在折叠
关闭当前折叠 zc 折叠光标所在区域
切换当前折叠 za 在展开和关闭之间切换
递归打开当前折叠 zO 展开当前区域内所有嵌套折叠
递归关闭当前折叠 zC 关闭当前区域及其嵌套折叠
打开所有折叠 zR 展开文件中的所有折叠
关闭所有折叠 zM 关闭文件中的所有折叠
减少折叠级别 zr 逐级展开折叠
增加折叠级别 zm 逐级关闭折叠
删除当前折叠 zd 删除光标所在的手动折叠
删除全部折叠 zE 删除文件中全部手动折叠

10. 实用组合命令

目标 命令 说明
删除 5 行 5dd 从当前行开始删除 5 行
复制 3 行 3yy 从当前行开始复制 3 行
向下移动 10 行 10j 向下移动 10 行
删除引号内内容 di" 删除双引号中的内容，保留引号
删除包括引号的内容 da" 删除内容及两侧双引号
修改括号内内容 ci( 清空圆括号内容并进入插入模式
复制大括号内内容 yi{ 复制大括号内部内容
选择当前单词 viw 进入可视模式并选择当前单词
删除当前段落 dap 删除当前段落
复制当前段落 yap 复制当前段落
格式化整个文件 gg=G 从文件开头格式化到文件末尾
删除全文 ggdG 删除文件中的全部内容
复制全文 ggyG 复制文件中的全部内容
全文替换并确认 :%s/old/new/gc 逐个确认替换结果

11. 推荐的基础配置

将以下内容写入 ~/.vimrc：

" 显示行号
set number
set relativenumber

" 高亮当前行和匹配括号
set cursorline
set showmatch

" 搜索设置
set ignorecase
set smartcase
set incsearch
set hlsearch

" 缩进设置
set expandtab
set tabstop=4
set softtabstop=4
set shiftwidth=4
set autoindent
set smartindent

" 显示与交互
set ruler
set wildmenu
set mouse=a
set encoding=utf-8

" 启用语法高亮和文件类型检测
syntax enable
filetype plugin indent on
