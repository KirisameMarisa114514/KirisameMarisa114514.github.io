Vim常用快捷键及命令手册

1. 模式切换

进入不同模式

模式 快捷键 说明
命令模式 Esc 退出其他模式，进入命令模式
插入模式 i 进入插入模式
可视模式 v 进入可视模式
可视行模式 V 进入可视行模式

退出不同模式

模式 快捷键 说明
命令模式 Esc 退出其他模式，进入命令模式
插入模式 Esc 退出插入模式，进入命令模式
可视模式 Esc 退出可视模式，进入命令模式
可视行模式 Esc 退出可视行模式，进入命令模式
可视块模式 Esc 退出可视块模式，进入命令模式

2. 常用命令

光标移动

命令 快捷键 说明
移动到行首 0 移动到行首
移动到行尾 $ 移动到行尾
移动到文件开头 gg 移动到文件开头
移动到文件结尾 G 移动到文件结尾
向上移动一行 k 向上移动一行
向下移动一行 j 向下移动一行
向上移动半页 Ctrl + b 向上移动半页
向下移动半页 Ctrl + f 向下移动半页
向上移动一页 PageUp 向上移动一页
向下移动一页 PageDown 向下移动一页
向上移动指定行数 Ctrl + g 向上移动指定行数
向下移动指定行数 Ctrl + g 向下移动指定行数

文本编辑

命令 快捷键 说明
插入文本 i 在光标前插入文本
替换文本 r 替换光标下的一个字符
替换行 R 替换整行文本
删除字符 x 删除光标下的一个字符
删除行 dd 删除光标所在的行
删除单词 dw 删除光标所在的单词
删除到行尾 d$ 删除光标到行尾的文本
删除到行首 d0 删除光标到行首的文本
删除到文件开头 dgg 删除光标到文件开头的文本
删除到文件结尾 dG 删除光标到文件结尾的文本
复制行 yy 复制光标所在的行
复制单词 yw 复制光标所在的单词
复制到行尾 y$ 复制光标到行尾的文本
复制到行首 y0 复制光标到行首的文本
复制到文件开头 ygg 复制光标到文件开头的文本
复制到文件结尾 yG 复制光标到文件结尾的文本
粘贴文本 p 在光标后粘贴文本
粘贴文本到行首 P 在光标前粘贴文本
撤销操作 u 撤销上一次操作
重做操作 Ctrl + r 重做上一次撤销的操作
查找文本 / 查找文本，按 Enter 键向下查找，按 N 键查找下一个匹配项
替换文本 :%s/old/new/g 将文件中的所有 old 替换为 new
替换指定行文本 :s/old/new/g 将当前行中的所有 old 替换为 new
替换指定范围文本 :start,end s/old/new/g 将指定范围内的所有 old 替换为 new

文件操作

命令 快捷键 说明
打开文件 :e filename 打开文件 filename
保存文件 :w 保存当前文件
另存为文件 :w filename 将当前文件另存为 filename
退出 Vim :q 退出 Vim
强制退出 Vim :q! 强制退出 Vim
保存并退出 Vim :wq 保存当前文件并退出 Vim
强制保存并退出 Vim :wq! 强制保存当前文件并退出 Vim
分屏打开文件 :sp filename 分屏打开文件 filename
水平分屏打开文件 :vsp filename 水平分屏打开文件 filename
关闭当前窗口 :q 关闭当前窗口
关闭所有窗口 :qa 关闭所有窗口
切换窗口 Ctrl + w 切换窗口
切换到上一个窗口 Ctrl + w + h 切换到上一个窗口
切换到下一个窗口 Ctrl + w + l 切换到下一个窗口
切换到最左边的窗口 Ctrl + w + j 切换到最左边的窗口
切换到最右边的窗口 Ctrl + w + k 切换到最右边的窗口
切换到最上边的窗口 Ctrl + w + j 切换到最上边的窗口
切换到最下边的窗口 Ctrl + w + k 切换到最下边的窗口
切换到指定编号的窗口 Ctrl + w + n 切换到指定编号的窗口
切换到指定名称的窗口 Ctrl + w + n 切换到指定名称的窗口

其他命令

命令 快捷键 说明
显示行号 :set number 显示行号
隐藏行号 :set nonumber 隐藏行号
显示状态栏 :set ruler 显示状态栏
隐藏状态栏 :set noruler 隐藏状态栏
显示光标位置 :set cursorline 显示光标位置
隐藏光标位置 :set nocursorline 隐藏光标位置
显示行尾符号 :set showendofline 显示行尾符号
隐藏行尾符号 :set noshowendofline 隐藏行尾符号
显示空白字符 :set list 显示空白字符
隐藏空白字符 :set nolist 隐藏空白字符
显示匹配括号 :set showmatch 显示匹配括号
隐藏匹配括号 :set noshowmatch 隐藏匹配括号
显示搜索结果 :set hlsearch 显示搜索结果
隐藏搜索结果 :set nohlsearch 隐藏搜索结果
显示语法高亮 :syntax on 显示语法高亮
隐藏语法高亮 :syntax off 隐藏语法高亮
显示缩进线 :set indentline 显示缩进线
隐藏缩进线 :set noindentline 隐藏缩进线
显示折叠 :set foldenable 显示折叠
隐藏折叠 :set folddisable 隐藏折叠
打开折叠 za 打开折叠
关闭折叠 zc 关闭折叠
切换折叠 zA 切换折叠
切换折叠 zC 切换折叠
打开所有折叠 zR 打开所有折叠
关闭所有折叠 zM 关闭所有折叠
切换所有折叠 zr 切换所有折叠
切换所有折叠 zm 切换所有折叠
跳转到指定行 :n 跳转到第 n 行
